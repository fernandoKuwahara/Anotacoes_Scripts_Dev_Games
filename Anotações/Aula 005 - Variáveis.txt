O QUE SÃO VARIÁVEIS?
	- Variáveis São Espaços Na Memória Que Tem A Função De Armazenar Diferentes Tipos De Dados.

TIPOS DE DADOS:
	- Int: O Tipo Inteiro Armazena Valores Comuns Por Exemplo, {1, 2, 3, 4, 5, ..., 100, 102}.
	- Float: O Tipo Float Armazena Valores Reais Por Exemplo, {1.1, 1.2, 1.3, 1.4, 1.5, ..., 100.2, 100.3}.
	- Bool: O Tipo Bool, Armazena Valores Booleanos Por Exemplo, {true, false}.
	- String: O Tipo String, Armazena Cadeia De Caracteres Por Exemplo, {"Olá Mundo", "Meu Primeiro Programa"}.

TIPOS DE DADOS DA UNITY:
	- GameObject: Tipo De Dado Usado Para Referênciar Um GameObject* Via Código.
	- Transform: Tipo De Dado Usado Para Referênciar O Transform* Via Código.

OBS:*
   Referência Via Código Depende Do Tipo De Dado Que Está Sendo Trabalhado No Momento.
